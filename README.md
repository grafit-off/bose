1. Responsive layout for different devices
2. Pixel Perfect
4. Semantic and Accessible Markup
5. High-Performance Scores on Google PageSpeed

HTML, BEM, CSS, SCSS, Pixel Perfect, Figma, Gulp, JavaScript
